METADATA INFORMATION
================================================================================

Each message in the enhanced HTML includes a "changes" field with an array
showing exactly what was enhanced. Examples:

  [["phrase", "release cycles"], ["close_rotation", "A -> B"]]
  → Phrase injected AND close pattern rotated

  [["phrase", "test maintenance"]]
  → Only phrase injection applied

  [["sentence_split", "Long sentence text..."]]
  → Only sentence splitting applied

  []
  → No changes (message was already optimal)

This metadata allows you to:
1. Track which enhancement types were applied to each message
2. Reverse engineer changes if needed
3. Analyze effectiveness of each enhancement type
4. Build A/B tests comparing enhancement types
