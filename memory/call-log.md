# Call Activity Log

**Purpose:** Weekly manual call tracking. Rob fills this in during/after each calling session. The weekly-analytics task reads this every Friday to include call metrics in the performance report.

**DB:** `analytics/outreach.db` → `call_activity` table (auto-synced by weekly-analytics task)

---

## How to Log Calls

After any calling session, add an entry below using this format:

```
### [DATE] — [SESSION NOTES]
- Dials: [N]
- Connects: [N] (someone actually picked up)
- Voicemails: [N]
- Meetings booked: [N]
- Notable connects: [name @ company — outcome in 1 line]
```

No need to be precise on exact times. End-of-day or end-of-week logging is fine.
